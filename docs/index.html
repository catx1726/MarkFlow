📥 详细安装指南 (本地打包文件)

🚀 Chrome & Edge (Chromium)

目前可通过打包文件安装，待发布至 Chrome Web Store 后，可实现一键安装。

  1. 下载最新的 highlight-mark-flow-chromium.zip 文件。
  2. 访问浏览器扩展管理页面：chrome://extensions。
  3. 开启右上角的 开发者模式。
  4. 点击 加载已解压的扩展程序，并选择解压后的文件夹（或直接拖拽 .zip 文件到页面）。

🦊 Firefox (临时加载)

目前通过临时加载安装（仅在当前会话有效），待发布至 Firefox Add-ons 后，可实现永久安装。

  1. 下载最新的 highlight-mark-flow-firefox.xpi 文件。
  2. 访问附加组件管理页面：about:addons。
  3. 点击设置图标（⚙️）-> 调试附加组件...
  4. 点击 临时加载附加组件...，选择 .xpi 或 .zip 文件进行安装。

注意： 临时加载的扩展程序会在浏览器重启后消失，如需永久安装请等待官方商店发布。
