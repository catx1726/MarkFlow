Mark Flow 提供的核心辅助功能

主要工具

知识浮岛：侧边栏实时管理

我们将所有高亮和备注集中显示在一个简洁的侧边面板中。您的笔记无需跳转，即可与原文同步显示，辅助您统一管理和阅读知识点。

提升便利性

便捷跳转，精确回顾

在侧边面板中点击任一标记，浏览器将快速滚动到该标记在原文中的位置。帮助您在知识点间实现高效的定位和穿梭。

数据安全

本地数据存储 & Markdown 导出

我们尊重您的数据所有权。所有数据均本地存储，安全无忧。支持一键导出 Markdown 文件，方便您导入 Obsidian、Notion 等主流笔记软件。

快速上手指南：三步管理您的知识

1

标记与添加备注

在任意网页上 选中 需要记录的文字。

*提示：您可以使用快捷键 ALT+S 快速标记，CTRL+C 复制选中内容。

2

开启知识浮岛

点击浏览器工具栏中的 Highlight Mark Flow 图标。侧边面板将即时显示当前页面所有已标记的内容。

*功能：您可以在浮岛中编辑、删除、导出笔记。

3

一键回到原文

在浮岛中点击任意笔记，浏览器主窗口将自动滚动到该标记在原文中的准确位置。

*优势：无需手动搜索和滚动，便捷定位知识源头。
