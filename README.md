## ✨ Highlight Mark Flow：网页高亮与知识回顾浮岛

告别在长文中的无限滚动。Highlight Mark Flow 将您最重要的知识点提取到侧边浮岛，实现**一键回顾与极速跳转**。

![Highlight Mark Flow](./assets/Highlight-Mark-Flow_2.gif '系统界面预览')

---

## 💡 解决的核心痛点

在阅读冗长复杂的学术论文、技术文档或深度报告时，回顾前文中的关键论点通常需要：

- 反复滚动屏幕或手动搜索关键词。
- 在多个标签页中切换笔记和原文。
- 使用功能复杂、需要付费或强制登录的传统工具。

**Highlight Mark Flow** 致力于提供一个 **轻量、免费、高效** 的解决方案，让您的阅读和回顾体验更加流畅。

## 核心功能与价值主张

Highlight Mark Flow 的核心价值在于建立一个 **“知识浮岛” (SidePanel)**，实现标记内容的**即时管理与回顾**。

### 🚀 极速跳转，精准定位

点击浮岛中的任何笔记或标记片段，即可**瞬间跳转**到原文的精确位置，告别手动搜索。

### 🗂️ 知识浮岛：侧边栏管理

所有标记和备注被集中显示在清晰、无干扰的侧边面板中。您无需离开阅读页面，即可：

- 浏览所有标记内容。
- 随时修改和添加备注。
- 根据颜色或备注内容快速过滤。

## 快速开始 (Quick Start)

### 1\. 标记内容

1. 选中网页上任何一段文字。
2. 在弹出的工具栏中选择您想要的颜色。

### 2\. 回顾与跳转

1. 点击浏览器工具栏上的 Highlight Mark Flow 图标，打开知识浮岛 (SidePanel)。
2. 浮岛会显示当前页面的所有标记和备注。
3. 点击列表中的任意条目，浏览器将自动滚动到原文对应位置。

## 🛠️ 技术栈 (Technical Stack)

Highlight Mark Flow 是一个基于现代 Web 技术的浏览器扩展，得益于以下核心工具的支持：

- **[Vitesse WebExt](https://github.com/antfu-collective/vitesse-webext):** 强大的 Web Extension 开发模板，提供快速开发和热模块替换 (HMR) 能力。
- **[Rangy](https://github.com/timdown/rangy):** 高效的跨浏览器 JavaScript 文本选择和范围操作库，确保高亮功能的稳定性和精确性。

## 📥 安装指南

### Chrome & Edge

即将发布到 Chrome Web Store。目前可通过打包文件安装：

1. 下载最新的 `highlight-mark-flow-chromium.zip` 文件。
2. 访问 `chrome://extensions`。
3. 开启 **开发者模式**。
4. 点击 **加载已解压的扩展程序**，选择解压后的文件夹（或直接拖拽 `.zip` 文件）。

### Firefox

即将发布到 Firefox Add-ons。目前可通过打包文件安装：

1. 下载最新的 `highlight-mark-flow-firefox.xpi` 文件。
2. 访问 `about:addons`。
3. 点击设置图标（⚙️）-\> **调试附加组件...** -\> **临时加载附加组件...**
4. 选择 `.xpi` 或者 `.zip` 文件进行安装。
