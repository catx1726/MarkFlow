## Highlight Mark Flow：网页高亮与知识回顾辅助

致力于优化您在长文阅读中的回顾体验。Highlight Mark Flow 将您标记的关键知识点集中提取到侧边浮岛，提供**便捷回顾与精确跳转**功能。

---

## 我们关注的阅读痛点

在处理冗长复杂的学术论文、技术文档或深度报告时，高效回顾前文中的关键信息可能面临挑战：

- 反复滚动屏幕或需要耗费时间手动搜索关键词。
- 笔记和原文分散在不同界面，造成回顾流程中断。
- 许多现有工具功能较重，可能需要付费或强制注册登录。

**Highlight Mark Flow** 旨在提供一个 **轻量、免费、专注于本地存储** 的解决方案，帮助您的阅读和回顾过程更加顺畅。

## 核心功能与价值定位

Highlight Mark Flow 的核心价值在于建立一个 **“知识浮岛” (SidePanel)**，辅助用户实现标记内容的**即时管理与便捷回顾**。

### 便捷跳转，精确回溯

点击浮岛中的任一笔记或标记片段，即可**快速跳转**到原文的精确位置，有效减少手动查找的时间。

### 知识浮岛：侧边栏整合管理

所有标记和备注会被清晰地集中显示在简洁的侧边面板中。您无需离开当前的阅读页面，即可完成以下操作：

- 轻松浏览当前页面的所有标记内容。
- 随时对标记添加或修改备注信息。
- 根据自定义颜色或备注内容进行快速筛选。

| 特性 | 详细 | 价值说明 |
| :-- | :-- | :-- |
| **完全免费、本地存储** | 所有数据存储在您的本地浏览器中，无需注册或付费。 | 低成本投入，提供高隐私保护的知识管理辅助。 |

| **轻量与直观** | 界面设计简洁，操作逻辑友好，力求减少传统工具中常见的复杂菜单。 | 上手门槛低，几乎没有学习成本。 |

## 快速开始 (Quick Start)

### 1\. 标记内容

1. 选中网页上任何一段文字。
2. 在弹出的工具栏中选择您想要的标记颜色。

### 2\. 回顾与跳转

1. 点击浏览器工具栏上的 Highlight Mark Flow 图标，打开知识浮岛 (SidePanel)。
2. 浮岛会显示当前页面的所有标记和备注。
3. 点击列表中的任意条目，浏览器将自动滚动到原文对应位置。

## 技术栈 (Technical Stack)

Highlight Mark Flow 是一个基于现代 Web 技术的浏览器扩展，得益于以下核心工具的支持：

- **[Vitesse WebExt](https://github.com/antfu-collective/vitesse-webext):** 强大的 Web Extension 开发模板，提供快速开发和热模块替换 (HMR) 能力。
- **[Rangy](https://github.com/timdown/rangy):** 高效的跨浏览器 JavaScript 文本选择和范围操作库，确保高亮功能的稳定性和精确性。

## 安装指南

### Chrome & Edge

即将发布到 Chrome Web Store。目前可通过打包文件安装：

1. 下载最新的 `highlight-mark-flow-chromium.zip` 文件。
2. 访问 `chrome://extensions`。
3. 开启 **开发者模式**。
4. 点击 **加载已解压的扩展程序**，选择解压后的文件夹（或直接拖拽 `.zip` 文件）。

### Firefox

1. 访问 [Firefox 应用商店](https://addons.mozilla.org/zh-CN/firefox/addon/highlight-mark-flow/)
2. 点击 “添加”
